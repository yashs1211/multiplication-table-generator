# 🚀 Multiplication Table Generator

A simple and interactive web application that generates multiplication tables based on user input.

## 🌐 Live Demo
👉 https://multiplication-tablegenerator.netlify.app/

---

## 📌 Features
- Enter any number to generate its multiplication table
- Displays results from 1 to 10 instantly
- Supports Enter key for quick interaction
- Clean and responsive user interface
- Click on table to reset input and output

---

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript (DOM Manipulation)

---

## ⚙️ How It Works
1. User enters a number in the input field
2. Clicks the **"Get Table"** button or presses **Enter**
3. JavaScript dynamically generates the multiplication table
4. Results are displayed instantly without page reload
